Laravel Fawry
==============

# Installation

Install via composer  
```bash
composer require maherelgamil/laravel-fawry

```

And then publish config

```bash
php artisan vendor:publish --tag="fawry-config"
```

Add `merchant_code` , `security_key` that's provided from Atfawry account

Now, Run migration

```bash
php artisan migrate
```
